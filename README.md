# AAR Corp (aar)
AAR Corp is a global aerospace and defense aftermarket solutions company and the number one aviation maintenance, repair, and overhaul (MRO) provider in North America. They offer aircraft parts supply, airframe and component MRO, integrated software solutions (Trax, Aerostrat, Airinmar), and expeditionary services to commercial airlines and government customers worldwide.
